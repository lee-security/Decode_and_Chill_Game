# ☕ Decode & Chill (로파이 디버거)

> **"결과를 보고 원인을 역추적한다"**  
> 복잡하고 차가운 리버싱(Reverse Engineering)의 개념을 따뜻한 파스텔톤 UI와 잔잔한 Lo-Fi 음악 속에서 쉽고 재미있게 배우는 힐링 인터랙티브 퍼즐 게임입니다.

---

## 🌟 주요 특징 (Key Features)

* **Chill & Cozy UI/UX**: 매트릭스풍의 어두운 해커 감성을 탈피하고, 눈이 편안한 파스텔 톤과 감성적인 디자인을 채택했습니다.
* **Ambient Lo-Fi BGM**: 몰입을 돕는 잔잔한 신디사이저 배경음악과 함께 편안한 환경에서 문제를 해결할 수 있습니다. (볼륨 조절 기능 포함)
* **적재적소 힌트 시스템**: 힌트 버튼과 안내 봇이 스크롤 영역 하단에 고정되어 있어, 흐름을 깨지 않고 자연스럽게 가이드를 제공받을 수 있습니다.

---

## 🎮 5단계 레벨 디자인 (Level Design)

게임은 리버싱의 핵심 개념을 직관적인 메타포로 치환한 5개의 스테이지로 구성되어 있습니다.

| 단계 | 스테이지명 | 학습 핵심 개념 (Reversing Core) | 미션 요약 |
| :--- | :--- | :--- | :--- |
| **🟢 1단계** | **블랙박스 열어보기** | 입력과 출력의 관계, `if/else` 조건문 추론 | 투시 돋보기를 통해 내부 조건을 확인하고 상자 열기 |
| **🟡 2단계** | **숨겨진 비밀번호 낚아채기** | 스트링(Strings) 탐색, 메모리 내 상수 비교 | 데이터 서랍장(메모리 뷰어)에서 실제 정답 텍스트 찾아내기 |
| **🟠 3단계** | **거짓말쟁이 기계 속이기** | 조건 분기 조작, 바이트 패치 (`Patching / NOP`) | 선로 스위치를 강제로 바꾸거나 장애물을 지워 승인 역으로 이동 |
| **🔴 4단계** | **컨베이어 벨트의 비밀** | 레지스터(`RAX`, `RBX`)와 기본 어셈블리 연산 | 연산 규칙을 역추적하여 원하는 결과가 나오기 위한 시작값 계산 |
| **🟣 5단계** | **마스터 디버거** | 중단점(`Breakpoint`), 실시간 메모리 조작 | 시간 제어 리모컨으로 시간을 멈추고 한 프레임씩 재생하며 취약점 탐색 |

---

## 🛠 기술 스택 (Tech Stack)

* **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
* **Audio Library**: `Tone.js` (실시간 브라우저 앰비언트 사운드 합성)
* **Icons**: FontAwesome / Google Material Icons

---

## 📱 UI/UX 개선 사항 (UI/UX Improvements)

* **레이아웃 고정 및 자동 스크롤**: 힌트 내비게이션 버튼을 화면 하단에 고정하고, 새로운 힌트 텍스트가 출력될 때마다 최신 메시지 위치로 화면이 부드럽게 자동 스크롤됩니다.
* **오디오 컨트롤러**: 사용자의 청각적 편안함을 위해 BGM 재생 시 실시간 볼륨 조절 슬라이더가 활성화됩니다.
